# 欢迎使用 LUI 组件库

✨ LUI 使用**Typescript**开发的 React 常用组件库，组件富含全面的**单元测试**，使用**travis**进行持续集成，主要用于中后台产品。

[**Github 开源地址**](https://github.com/licop/LUI)

<div className="subheading">安装</div>

组件库已发布到**npm**，推荐使用**npm**方式安装

```
  npm install @licop_pj/lui --save
```

引入 css

```
  import '@licop_pj/lui/dist/antd.css'
```
